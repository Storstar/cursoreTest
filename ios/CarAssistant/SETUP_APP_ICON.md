# Инструкция по установке иконки приложения

## Шаги для установки иконки:

1. **Подготовьте изображение:**
   - Откройте ваше изображение в графическом редакторе
   - Измените размер до **1024x1024 пикселей**
   - Сохраните в формате PNG
   - Назовите файл: `AppIcon.png`

2. **Добавьте иконку в проект:**
   - Откройте Xcode
   - В навигаторе проекта найдите: `CarAssistant/Resources/Assets.xcassets/AppIcon.appiconset/`
   - Перетащите файл `AppIcon.png` в эту папку
   - Или: Правой кнопкой на `AppIcon.appiconset` → "Show in Finder"
   - Скопируйте `AppIcon.png` в эту папку

3. **Проверьте в Xcode:**
   - Откройте `AppIcon.appiconset` в Xcode
   - Убедитесь, что иконка отображается в слоте 1024x1024

4. **Очистите и пересоберите проект:**
   - Product → Clean Build Folder (Cmd+Shift+K)
   - Product → Build (Cmd+B)

5. **Проверьте иконку:**
   - Запустите приложение на симуляторе или устройстве
   - Иконка должна отображаться на главном экране

## Требования к иконке:

- **Размер:** 1024x1024 пикселей
- **Формат:** PNG (без прозрачности для iOS)
- **Цветовое пространство:** sRGB
- **Фон:** Рекомендуется непрозрачный фон

## Примечание:

Если вы хотите использовать изображение с прозрачностью, iOS автоматически добавит черный фон.
Для лучшего результата используйте иконку с непрозрачным фоном.
